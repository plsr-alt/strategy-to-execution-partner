## 1. 統合コンセプト

4つのPJは **「1つのAIコンテンツビジネスの4つの出力チャネル」** である。

```
[共通エンジン]
  FLUX.2 [klein] 4B (画像生成・Apache 2.0・ローカル)
  + LoRA (キャラクター一貫性)
  + Groq LLM (テキスト生成・無料枠)
       │
       ├─→ デジタル販売（BOOTH/FANBOX/DLsite）  ← 即収益
       ├─→ ショート動画量産（Ken Burns+FFmpeg）  ← 自動化
       ├─→ マルチチャネルSNS（IG/Pinterest API） ← 配信
       └─→ AIインフルエンサー Rena（ブランド）    ← ブランド構築
```

**全ツール無料構成** — fal.ai/Pika/Ayrshare 不要に変更済み。
X API は有料化済み（2026/2/7〜）→ 全PJで手動投稿。

## 4. 市場調査サマリー

### AIインフルエンサー市場（`04_RESEARCH/2026-03-07_ai_influencer_market_survey.md`）
- **グローバル市場**: 69.5億ドル（約1兆700億円）
- **トップ事例**: Aitana López（月$300-500K）、Lil Miquela（月$200-300K）
- **日本市場**: 初期段階 = 参入チャンス大
- **テック×LS評価**: S-tier（グリーン）— 最も参入障壁が低く成長余地大
- **AI開示の影響**: 短期-10〜15%エンゲージメント低下 → 長期回復

### デジタル販売市場（`04_RESEARCH/2026-03-07_ai_digital_sales_market_survey.md`）
- **市場規模**: 100〜300億円/年（成長率40〜50%/年）
- **プラットフォーム手数料**: BOOTH 5.6% << FANBOX 10% << DLsite 30〜50%
- **成功事例**: きゃんでぃ（月10万円/3ヶ月）、FANBOX AI（月40万円/2ヶ月）
- **売れ筋**: キャラクターイラスト > 壁紙 > 3Dモデル > アニメCG > ゲーム素材
- **勝ちパターン**: キャラ一貫性(FLUX.2) + 複合販売 + SNS連携
