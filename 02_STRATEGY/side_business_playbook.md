# サイドビジネス Playbook — 最初の1円を作る

## 0. 立ち上げ設計（最初に決めるルール）

**目的**: 月100万ではなく「最初の売上1円」を作る。

### コスト設計（固定費を潰す）

| 項目 | 上限 |
|------|------|
| 初期費用 | 0〜5,000円 |
| 固定費 | 月0〜1,000円 |
| Web | 無料テンプレで1日公開 or なくてもOK |
| 名刺 | 作らない（SNSプロフィールで代替） |
| 拠点 | 自宅 |
| 広告 | 0円 |
| ツール | AI無料枠＋note（必要最低限） |

### 生活防衛ルール（失敗しても死なない）

- 本業収入を維持
- 生活費6ヶ月分確保
- 1案件（1トライ）の投資上限：1万円以内

---

## 1. 事業の勝ち方（思想）

- 売上0円：改善不能
- 売上1円：改善開始
- だから「当てにいく」より **小さく試して早く学ぶ**
- 基本戦略：**小さく参入／小さく撤退**

---

## 2. 最短で1円を作る型（AI × note販売）

### この型を採用する理由

| 条件 | 評価 |
|------|------|
| 在庫 | 0 |
| 固定費 | ほぼ0 |
| 制作期間 | 1日〜 |
| 即販売 | 可能 |
| 撤退コスト | 0 |

---

## 3. 実行手順（5日で販売開始の標準プロセス）

### Day1：テーマ出し（AI）
- "誰のどんな悩みを、何で解決するか" を10個出す
- その中から「自分が書ける」×「需要がありそう」を2つ残す

### Day2：構成（60分）
noteの型（テンプレ）に落とす：
1. 読者の悩み（共感）
2. よくある失敗
3. 解決の全体像（結論）
4. 手順（チェックリスト）
5. 実例（できれば）
6. よくある質問
7. まとめ＋次アクション

### Day3：本文（2〜3時間）
- AIで下書き → 自分の経験・言葉で整える
- 「読者が今日動ける」粒度まで落とす（手順化）

### Day4：公開（note）
- タイトルだけは時間をかける（最重要）
- 冒頭で「誰向けか」を明確にする
- 価格は小さく（例：500円〜）でOK

### Day5：告知（X）
- 告知は1回で終わらせない（最低3回）
  - 投稿①：悩みの代弁＋結論
  - 投稿②：手順の一部を無料公開
  - 投稿③：購入後の変化（ベネフィット）

---

## 4. 検証 → 改善の回し方（コア）

### 最低ライン（例）
- フォロワー300人
- note 500円
- 購入率1% → 3本売れて1,500円
- → この「最初の売上」が出た時点で勝ち（改善フェーズへ）

### 改善レバー（売上が出たら必ずやる）
1. **タイトル修正**（CTR改善）
2. **導線修正**（プロフィール、固定ポスト、告知文）
3. **内容改善**（読者の「買った後の満足度」）

### 改善サイクルの期待値
- 2本目：5〜10本
- 3本目：20本前後
- → 3記事目で1万円ラインが現実的

---

## 5. 撤退基準（迷わないためのルール）

| 条件 | 判定 |
|------|------|
| 2週間やって反応0 | 撤退 |
| 3週間で閲覧100未満 | 撤退 |
| 1ヶ月売上0 | テーマ変更 |

> 撤退＝失敗じゃなく **学習の完了**。次の弾へ。

## 7. 詰まりポイント別の処方箋

| 詰まり | 処方箋 |
|--------|--------|
