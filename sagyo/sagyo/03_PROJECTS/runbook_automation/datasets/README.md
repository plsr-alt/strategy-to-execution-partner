# datasets/ — 取り扱い注意

⚠️ **このディレクトリにはPIIが含まれる可能性があります。**

## 取り扱いルール

- [ ] PII項目（氏名・住所・口座番号等）は必ずマスキングしてから保存
- [ ] このディレクトリはGitにコミットしない（`.gitignore`済み）
- [ ] S3保存時はSSE-KMS暗号化必須
- [ ] アクセスは指定IAMロールのみ
- [ ] 保管期間: PoC終了後30日で完全削除

## ディレクトリ構成（案）

```
datasets/
├── raw/          ← 元データ（ローカルのみ保持、S3 or ネットワーク非公開）
├── masked/       ← マスキング済みデータ（OCR処理に使用）
├── ground_truth/ ← 正解データ（評価用）
└── test_set/     ← 評価セット（50件固定）
```
