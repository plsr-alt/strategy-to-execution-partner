# TOEIC パート別 必勝攻略ガイド

> 850点を狙うための、全7パートの効率的な解法とテクニック集

---

## 📋 全体の時間配分（リーディング75分）

| パート | 問題数 | 目標時間 | 1問あたり |
|--------|--------|---------|----------|
| Part 5 | 30問 | **10分** | 20秒 |
| Part 6 | 16問 | **10分** | 37秒 |
| Part 7 | 54問 | **55分** | 約1分 |

> ⚠️ Part 5, 6 を「いかに速く正確に抜けるか」がPart 7の得点を左右する。

---

## 🎧 Listening Section（約45分・100問）

---

### Part 1：写真描写問題（6問）

**形式**：写真1枚に対して、音声で流れる4つの描写文から正しいものを選ぶ。

#### 🏆 必勝ポイント
1. **ディレクション中に6枚全ての写真を見ておく**
   - 人物の動作？モノの配置？風景？を事前に把握。
2. **消去法が最強の武器**
   - 正解を「探す」のではなく、不正解を「消す」。明らかに違うものを瞬時に切り捨てる。
3. **「状態」と「動作」を区別する**
   - `wearing`（着ている状態） vs `putting on`（着る動作）は超頻出のひっかけ。
4. **受動態の進行形に注意**
   - `is being repaired`（修理されている最中）と `has been repaired`（修理済み）は別物。
5. **似た発音の単語で騙されない**
   - 例：`copying`（コピーする）と `coffee`（コーヒー）。音だけで飛びつかない。

#### ⚡ 解き方の流れ
```
写真を見る → 主なモノ・人物を把握 → 音声を聞く → 消去法で絞る → マーク
```

---

### Part 2：応答問題（25問）

**形式**：1つの質問・発言に対して、3つの応答から最も適切なものを選ぶ。**問題も選択肢も印刷されていない（音声のみ）。**

#### 🏆 必勝ポイント
1. **冒頭の1〜3語（特に疑問詞）を死守する**
   - `Who` → 人を答える選択肢
   - `When` → 時間を答える選択肢
   - `Where` → 場所を答える選択肢
   - **疑問詞を聞き逃すと、その問題は終わる。**
2. **Yes/Noで答えない「間接応答」が正解になることが多い**
   - Q: "Do you know where the meeting room is?"
   - ✅ "It's on the third floor."（直接回答ではない間接応答が正解）
3. **似た音の単語トラップを見抜く**
   - 質問文にある単語と発音が似ている選択肢は、大抵ひっかけ。
4. **質問文の単語がそのまま出てきたら疑う**
   - 同じ単語を繰り返す選択肢は不正解のサインであることが多い。

#### ⚡ 解き方の流れ
```
冒頭の疑問詞を聞く → 質問の種類を特定 → 3つの応答を聞く → 消去法 → マーク
```

---

### Part 3：会話問題（39問）

**形式**：2〜3人の会話を聞き、設問3つに答える。設問と選択肢は問題用紙に印刷されている。

#### 🏆 必勝ポイント
1. **「先読み」がすべてを決める**
   - 音声が流れる前に、3つの設問 + 選択肢に目を通す。
   - 「誰が・どこで・何について」話しているかを予測する。
2. **先読みは「次の1セット（3問）だけ」に集中**
   - 欲張って先まで読むと混乱する。次の3問だけ。
3. **設問の順番 ≒ 会話の順番**
   - 設問1のヒントは会話の前半、設問3は後半にある。順番に聞けばOK。
4. **図表問題は先に図を読む**
   - スケジュール表、地図、価格表などが出たら、図の内容を先に把握。
5. **聞き取れなくても引きずらない**
   - 1問落としても次の先読みに切り替える。引きずると連鎖で落とす。

#### ⚡ 先読みのタイミング
```
前の問題のマーク → 即座に次の3問を先読み → 音声開始 → 聞きながら解答
```

---

### Part 4：説明文問題（30問）

**形式**：1人のスピーカーによる説明文（留守電、アナウンス、広告など）を聞き、設問3つに答える。

#### 🏆 必勝ポイント
1. **Part 3と同じ「先読み」戦略が有効**
   - 設問から「何のアナウンスか」「誰に向けたものか」を予測。
2. **よく出るジャンルを把握しておく**
   - 留守番電話、会議の冒頭挨拶、ラジオ広告、社内アナウンス、ツアーガイド
3. **「意図問題」に注意**
   - "What does the speaker imply?" → 直接言っていないことを推測する問題。
   - 話者のトーンや文脈から判断する。
4. **冒頭の1〜2文で場面設定を掴む**
   - 最初の文はほぼ必ず「誰が、何のために話しているか」を示す。

## 📖 Reading Section（75分・100問）

---

### Part 5：短文穴埋め問題（30問）

**形式**：短い英文の空欄に入る最も適切な単語・語句を選ぶ。

#### 🏆 必勝ポイント
1. **まず選択肢を見て問題タイプを判別する（2秒）**
   - **品詞問題**：語尾だけ違う（例：`carefully / careful / care / caring`）→ **文法構造で解く**
   - **語彙問題**：全く違う単語が並ぶ → **文意で解く**
   - **文法問題**：時制・接続詞・前置詞など → **文法ルールで解く**
2. **品詞問題は空欄の前後だけで解ける（5〜10秒）**
   - 冠詞（a/the）の後 → **名詞**
   - 名詞の前 → **形容詞**
   - 動詞・形容詞を修飾 → **副詞**
   - 主語の後 → **動詞**
3. **全文を読まない勇気**
   - 品詞問題は前後2〜3語だけで解答可能。全文読むのは時間のムダ。
4. **迷ったら飛ばす（20秒ルール）**
   - 20秒考えてわからなければ、直感でマークして次へ。Part 7の時間を守る。

#### ⚡ 解き方の流れ
```
選択肢を見る(2秒) → 問題タイプ判別 → 品詞なら前後だけ / 語彙なら全文 → マーク
```

---

### Part 6：長文穴埋め問題（16問）

**形式**：ビジネス文書（メール、通知など）の4つの空欄を埋める。うち1問は「文挿入問題」。

#### 🏆 必勝ポイント
1. **全文を先にざっと読む（スキミング）**
   - 「誰が・誰に・何について」の文脈を掴んでから空欄を解く。
2. **Part 5と同じ解法が使える問題が約35%ある**
   - 品詞問題・文法問題は空欄の前後だけで解ける。これを先に片付ける。
3. **文挿入問題（1問/文書）の攻略**
   - **前後の文をよく読む**：挿入する文の前後との論理的つながりが手がかり。
   - **指示語に注目**：`this`, `these`, `such`, `it` → 前の文の何かを指す。
   - **接続詞に注目**：`however`（逆接）、`therefore`（結論）、`in addition`（追加）。
4. **文脈依存の問題は後回しにしてもOK**
   - 先に品詞・文法問題を解き、文の流れを掴んでから文挿入に取り組む。

#### ⚡ 解き方の流れ
```
全文スキミング → 品詞・文法問題を先に解く → 文挿入問題に取り組む → マーク
```

---

### Part 7：長文読解問題（54問）

**形式**：メール、記事、広告、チャット等の文書を読み、設問に答える。シングル / ダブル / トリプルパッセージの3種類。

#### 🏆 必勝ポイント
1. **「設問 → 本文」の順番で読む**
   - 先に設問を読んで「何を探すか」を明確にしてから本文に入る。
2. **設問の順番 ≒ 本文の流れ**
   - 設問1の答えは本文の上部、設問3の答えは下部にあることが多い。順番に探す。
3. **スキミング→スキャニングの2段階**
   - **スキミング**：全体をざっと読んで文書の種類・目的を把握（10秒）
   - **スキャニング**：設問のキーワードを本文から探し出す
4. **「言い換え」に敏感になる**
   - 本文：`postpone` → 選択肢：`delay` のように同じ意味を別の単語で表現。
5. **NOT問題・推測問題は後回し**
   - "Which is NOT mentioned?" は全選択肢を本文と照合する必要があるため時間がかかる。後に回す。
6. **ダブル・トリプルパッセージは関連箇所を探す**
   - 2〜3つの文書の「共通するキーワード（人名、日付、金額）」で関連箇所を特定する。

#### ⚡ 時間管理のコツ
```
1問1分を意識 → 難問は2分以内に切り上げ → 最後5分で見直し＆マーク漏れチェック
```

## 📝 パート別の優先学習順

850点を狙う場合の効率的な強化順：

1. **Part 5**（即効性が高い。文法と語彙の強化がそのまま点数に直結）
2. **Part 2**（短い音声なので対策しやすく、正答率を上げやすい）
3. **Part 1**（少ない問題数で満点が狙える）
4. **Part 3, 4**（先読みの技術を習得すれば一気に伸びる）
5. **Part 6**（Part 5の延長。文脈理解を加えるだけ）
6. **Part 7**（地力が必要。多読・速読の積み重ねがモノを言う）
