# TOEIC Part 5 文法完全攻略チートシート

> abceedが見られない時でも、これだけ読めば「文法の型」が身につく最強まとめ

---

## ⚡️ 秒速で解くための3つの鉄則
1. **まず選択肢を見る（2秒）**：品詞問題（語尾違い）か、意味問題（語彙違い）かを瞬時に判別。
2. **空欄の前後をチェック**：全文を読まずに解ける問題が4割。
3. **迷ったら「前置詞＋名詞」か「接続詞＋文」かを確認**：これが最大の得点源。

---

## 1. 品詞問題（語尾で決める）
TOEIC Part 5の最大の得点源。単語の意味を知らなくても「語尾」から品詞を選べば即答可能。

### 📌 空欄に入る品詞のルール
- **名詞が入る場所**：冠詞（a/the）の後、所有格（my/Ms. Tanaka's）の後、前置詞（in/on/about）の後。
- **形容詞が入る場所**：名詞の前、be動詞の後。
- **副詞が入る場所**：動詞・形容詞・副詞・文全体を修飾する場所。**「文が完成している（どこにも隙間がない）」なら副詞が正解。**

### 📌 覚えておくべき語尾（接尾辞）
- **名詞**：-ment, -tion, -ity, -ance, -ness, -ship
- **形容詞**：-tive, -sive, -ful, -less, -ous, -able, -ent, -ant
- **副詞**：**-ly**（形容詞+lyが基本）

---

## 2. 動詞の形（3つのチェック：単複・時制・態）
空欄が動詞の場合、以下の順序で選択肢を削る。

1. **主語との一致（単数・複数）**：主語が単数なら `s` がつく。
2. **態（能動態・受動態）**：
    - 後ろに「目的語（〜を）」がある → **能動態** (V)
    - 後ろに「目的語」がない / 前置詞（byなど）がある → **受動態** (be + Ved)
3. **時制のキーワード**：
    - `yesterday / ago / last week / once` → **過去形**
    - `tomorrow / next / soon` → **未来形**
    - `since / for (+数年) / so far` → **現在完了形 (have + Ved)**

---

## 3. 接続詞 vs 前置詞（意味は同じ・形が違う）
TOEICが最も好む「ひっかけ」ポイント。意味は同じでも、後ろの形が違う。

| 意味 | **接続詞 (+ S + V)** | **前置詞 (+ 名詞/Ving)** |
| :--- | :--- | :--- |
| 〜なので | **because / since / as** | **because of / due to / owing to** |
| 〜にも関わらず | **although / though / even though** | **despite / in spite of** |
| 〜の間 | **while** | **during** |

> **💡 裏技**：空欄の後ろに「〜ing（動名詞）」が来ていたら、前置詞が正解になる確率が高い。

---

## 4. 代名詞（表で覚える）
格の変化を問う問題が必ず出ます。

- 空欄 + 名詞 → **所有格** (`my` book / `their` office)
- 文末 / 動作の対象 → **目的格** (contact `me` / call `him`)
- 「自分自身で」 → **再帰代名詞** (`himself` / `themselves`)
- 空欄の後に何もなし（〜のもの） → **所有代名詞** (`ours` / `yours`)

---

## 5. 前致詞・接続詞の頻出セット（コロケーション）
これが見えたら0.1秒で即答できるセット。

- **both** A **and** B
- **either** A **or** B
- **neither** A **nor** B
- **not only** A **but also** B
- **whether** A **or** B
- **depend on** / **rely on**（〜に頼る）
- **comply with**（〜に従う）
- **participate in**（〜に参加する）

---

## 🚀 850点への「さらなる一歩」
- **関係代名詞**：空欄の前の名詞（先行詞）を見て判断。
    - **Who**：先行詞が人
    - **Which / That**：先行詞が物
    - **Whose**：所有の関係（〜の）
- **原形をとる動詞（仮定法現在）**：
    - `it is essential that S (should) 原形`
    - `recommend / suggest that S (should) 原形`
    - これらが出たら、たとえ主語が三人称単数でも動詞を「原形」のままにするのが正解。
