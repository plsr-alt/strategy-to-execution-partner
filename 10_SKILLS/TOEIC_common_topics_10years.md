# TOEIC 頻出シナリオ×必須語彙 完全攻略ガイド（過去10年分析）

TOEICで繰り返し出題される「お決まりのシチュエーション」と、そこで必ず使われる「キーワード」をセットでまとめました。これを頭に入れておくだけで、内容の理解度が飛躍的に高まります。

---

## � 1. 人事・福利厚生（Human Resources & Benefits）
**【シナリオ】** 新人の採用、社員の表彰、研修の案内、福利厚生の変更など。
- **よくある展開**：
    - 「優秀な社員を称えるパーティーを開くので参加して（でも予算が少し足りない）」
    - 「新しい研修プログラムを導入するから、担当者は今日中に申し込んで」

| 英単語 / 表現 | 意味 / ニュアンス |
| :--- | :--- |
| **Appreciation** | 感謝、高く評価すること。表彰式で頻出。 |
| **Engagement** | 従業員の関与・愛着心。 |
| **Onboarding** | 新人研修、入社プロセス。 |
| **Turnover** | 離職率。 |
| **Compensation** | 報酬、給与。 |
| **Performance review** | 人事考課、評価面談。 |
| **Acknowledge** | （功績などを）認める。 |

## 🛠 2. 設備維持・メンテナンス（Maintenance & Facilities）
**【シナリオ】** 空調の故障、オフィスの点検、備品の修理、工事による遅延。
- **よくある展開**：
    - 「エアコンが壊れたけど、業者が来るのは明日の朝になる」
    - 「ビルの点検で水が止まるので、あらかじめ汲んでおいて」

| 英単語 / 表現 | 意味 / ニュアンス |
| :--- | :--- |
| **Appliance** | 電化製品（冷蔵庫、電子レンジなど）。 |
| **Plumbing** | 配管、水道設備。 |
| **Inspection** | 検査、点検。 |
| **Out of order** | 故障中。 |
| **Under construction** | 工事中。 |
| **Formal request** | 正式な依頼（修理依頼などで使用）。 |
| **Inventory** | 在庫、備品リスト。 |

## 📈 3. マーケティング・販売（Marketing & Sales）
**【シナリオ】** 新製品の発売、ブランド認知度の向上、広告キャンペーンの成果。
- **よくある展開**：
    - 「新製品のキャンペーンを打った結果、売上が予想を上回った」
    - 「ターゲット層を絞り込むために、アンケート調査を実施しよう」

| 英単語 / 表現 | 意味 / ニュアンス |
| :--- | :--- |
| **Brand awareness** | ブランド認知度。 |
| **Cash rebate** | 現金払い戻し。実質の割引として登場。 |
| **Target audience** | ターゲット層。 |
| **Promotion** | 販促、宣伝。 |
| **Market research** | 市場調査。 |
| **Product launch** | 製品の発売、発表。 |
| **Consumer behavior** | 消費者行動。 |

## 💻 4. IT・デジタル・リモート（IT & Remote Work）
**【シナリオ】** オンライン会議の設定、システムトラブル、リモートワークへの移行。
- **よくある展開（最新トレンド）**：
    - 「明日のオンライン会議、リンクが見つからないんだけど」
    - 「セキュリティアップデートのため、今夜はサーバーにアクセスできません」

| 英単語 / 表現 | 意味 / ニュアンス |
| :--- | :--- |
| **Online conference** | オンライン会議（Webinarとも言う）。 |
| **Troubleshoot** | 問題を解決する、トラブルを特定する。 |
| **User-friendly** | 操作が簡単で使いやすい。 |
| **Attachment** | 添付ファイル（メールなどで頻出）。 |
| **Remote work** | リモートワーク。 |
| **Incompatible** | 互換性がない。 |
| **Software update** | ソフトウェアの更新。 |

## 🌱 5. ESG・サステナビリティ・社会貢献（ESG & CSR）
**【最新トレンド】** 環境保護、社会貢献、企業の社会的責任（CSR）に関する話題。
- **よくある展開（2024-2025年増加中）**：
    - 「わが社はプラスチック使用量を削減する新しい方針を打ち出しました」
    - 「地域の清掃ボランティアに参加する社員を募集しています」
    - 「サステナビリティに関するセミナーへの招待状が届く」

| 英単語 / 表現 | 意味 / ニュアンス |
| :--- | :--- |
| **Sustainability** | 持続可能性（サステナビリティ）。 |
| **Carbon emission** | 二酸化炭素排出。削減対象として頻出。 |
| **Corporate Social Responsibility (CSR)** | 企業の社会的責任。 |
| **Environmentally friendly** | 環境に優しい（エコな）。 |
| **Initiative** | 新しい取り組み、構想。 |
| **Implementation** | （新しい方針などの）実施、導入。 |
| **Biodiversity** | 生物多様性。 |

## ✈️ 6. 出張・物流・交通（Travel & Logistics）
**【シナリオ】** 旅程の変更、フライトの遅延、ホテルの予約、荷物の配送。
- **よくある展開**：
    - 「悪天候でフライトがキャンセルになった。別の便を探して」
    - 「注文した荷物が届かない。送り先を確認してほしい」

| 英単語 / 表現 | 意味 / ニュアンス |
| :--- | :--- |
| **Itinerary** | 旅程、旅行計画。 |
| **Reservation** | 予約（bookingとも言う）。 |
| **Complimentary** | 無料の。ホテルの朝食などで頻出。 |
| **Inclement weather** | 悪天候。遅延の理由の定番。 |
| **Delay / Postpone** | 遅らせる / 延期する。 |
| **Confirmation number** | 確認番号。 |

## � 知っておくと得する「言い換え（Paraphrasing）」リスト
TOEICは「本文の単語と、選択肢の単語をわざと変える」のが最大の特徴です。

- **Buy** → **Purchase**（購入する）
- **Job** → **Occupation / Position**（職業、職種）
- **Reservation** → **Arrangement**（手配、準備）
- **Complimentary** → **At no cost / Free of charge**（無料の）
- **Provide** → **Offer**（提供する）
- **Attendee** → **Participant**（参加者）
- **Check** → **Inspect / Examine**（検査する）

## 📌 学習のアドバイス
単語単体で覚えるのではなく、**「水道の工事（Plumbing inspection）だから、水が止まる（water will be shut off）」**のように、シナリオとセットで単語を脳に流し込むのが850点への近道です。
