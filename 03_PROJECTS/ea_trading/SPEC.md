# EA仕様書（SPEC.md）：MVP仕様定義～V1差分仕様

## 1. MVP仕様概要定義

| 項目 | MVP値 | 理由・設計背景 |
|-----|------|--------|
| **戦略概要** | ボリバン±2σ + RSI 30/70フィルター | レンジ相場に最適化。業界標準パラメータで過剰最適化リスク最小 |
| **適用足** | 1H足（1時間足） | 日中スプレッド安定・流動性充分・スリッページ予測可能 |
| **通貨ペア数** | 1ペア（USD/JPY） | MVP最小化・統計検証に集中。V1で3～5ペア対応へ拡張 |
| **バックテスト期間** | 12ヶ月（直近） | 相場サイクル4～6シーズン網羅・2000回取引目標達成 |
| **期待成績** | 勝率50～60% / Max DD ≤15% / PF 1.3～1.8 | 保守的な予測。リアルでは-35%成績低下を見込む |

## 5. ロット計算式と資金管理理論

### 5.1 固定リスク比率法

**ロット計算式**（MVP採用：固定リスク2%方式）

```
Lot = (口座残高 × リスク率%) ÷ (SL_pips × 1pip当たりの価値)

【具体例1】USD/JPY の場合（最重要）
  口座残高: 100万円
  リスク率: 2% → リスク許容額 = 100万 × 0.02 = 2万円

  USD/JPY仕様:
    1 Lot = 10万通貨
    1 pips = 1000円

  シナリオ：SL = 20pips と設定した場合
    損失額 = 20 pips × 1000円 = 2万円（リスク2%）
    Lot = 2万円 ÷ (20 × 1000) = 1.0 Lot
    → ポジションサイズ：10万通貨

【具体例2】複数取引での月利5%検証
  勝率60% / 平均利益100pips / 平均損失20pips の場合：
    利益トレード18回 × 100pips × 1000円 × 1.0Lot = 180万円
    損失トレード12回 × 20pips × 1000円 × 1.0Lot = 24万円
    純利益 = 156万円 → 初月100万円に対して月利156%

  実際の月利5%実現（ドローダウン・複数ペア分散考慮）：
    → 月間20～25回の取引と-35%乖離を見込むと月利5%が妥当
```

### 5.2 ケリー基準との関係（数学的根拠）

```
f* = (W × p - L × (1 - p)) ÷ W

where:
  f* = 最適投資比率（ケリー理論値）
  W = 平均利益（pips）
  L = 平均損失（pips）
  p = 勝率

計算例：勝率 60% / 平均利益 100pips / 平均損失 20pips
f* = (100 × 0.60 - 20 × 0.40) ÷ 100 = 0.52
→ 口座資金の52%をリスク配分すべき（理論値・危険）

実践対応（リサーチ16行目）：
  1/4 Kelly法：0.52 × 0.25 = 0.13（13%）
  MVP 固定2%方式：さらに保守的（0.02 = 2%）
  → フォワード崩壊リスク最小化
```

### 5.3 複利効果・複利計算実例

月利5%達成時の複利曲線：

| 月 | 口座残高 | リスク2% | SL20pips時Lot | 利益 | 複利後残高 |
|---|---------|---------|-------------|------|---------|
| 初月 | 100.0万 | 2.0万 | 1.00 | 5.0万 | 105.0万 |
| 2月 | 105.0万 | 2.1万 | 1.05 | 5.25万 | 110.25万 |
| 3月 | 110.25万 | 2.21万 | 1.10 | 5.51万 | 115.76万 |
| 6月 | 131.68万 | 2.63万 | 1.32 | 6.58万 | 138.28万 |
| 12月 | 179.08万 | 3.58万 | 1.79 | 8.95万 | **188.03万** |

**複利効果**：12ヶ月で100万円 → 188万円（88%増）。固定ロットでは60%増に留まる。

## 8. SL・TP設計の数学的根拠

### 8.1 SL設定：ATR × 2倍

**ATR（Average True Range）の特性**:
- 直近14足のボラティリティを数値化
- 相場静寂時→ATR小→SL狭い / 相場荒れ→ATR大→SL広い
- **適応性**：市場環境に自動調応

**ATR × 2倍 根拠**（リサーチ14行目）:
- **中期足での適正リスク**：1%前後
- **実績統計**：
  - トレンド型 15～25% MaxDD
  - レンジ型 10～15% MaxDD
- **倍数バランス**：
  - ATR × 1倍：SL狩られ多発
  - ATR × 2倍：**適度（採用）**
  - ATR × 3倍：リスク超過

**計算例（ドル円）**:
```
ATR = 20 pips → SL = 40 pips
口座100万、Lot=1.0時の損失 = 4万円（リスク4%）
→ リスク2%目標で Lot = 0.5 に自動調整

相場静寂時（ATR 10）：SL 20 pips → Lot 1.0（相対的に大ロット確保可能）
相場荒れ時（ATR 30）：SL 60 pips → Lot 0.33（相対的に小ロット）
→ ボラティリティに応じた自動調整

※ ロット「大」「小」は相対表記：ATRが小さい = ボラティリティ低 =
  リスク額が小さくても口座損失率が同じ2%となるため、Lotは相対的に大きくなる。
  この自動調整が固定ロット方式との差別点。
```

### 8.2 TP設定：20MA（ボリバン中心線）

**設計ポイント**:
1. **レンジ戦略特性**：反発トレードで中心線への回帰
2. **簡潔性**：指標1つで検証容易
3. **過剰最適化回避**：TP値複数試行なし

**代替案との比較**:

| TP設定案 | 利幅期待値 | 短所 | 採用判定 |
|---------|---------|------|--------|
| **20MA** | 20～40pips | やや小さい可能性 | **MVP推奨** |
| BB_Middle±5 | 20～35pips | 追加パラメータ | 非推奨 |
| 固定50pips | 全期間同じ | 環境非適応 | 非推奨 |
| RSI中点(50) | 20～50pips | ロジック複雑 | 非推奨 |

**採用理由**：MVP検証優先に複雑さ排除。業界標準ボリバン中心線を信頼。
