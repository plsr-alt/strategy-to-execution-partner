## 2. キャラクター一貫性の技術要件

### 2-1. LoRA学習手順（概要）

#### ステップ1: トレーニングデータセット準備
- **データセット**: キャラクターの写真 50-100枚（バリエーション付き）
  - 異なる角度: 正面、横顔、45度
  - 異なる照明: 昼光、スタジオ、自然光
  - 異なる背景: オフィス、スタジオ、屋外
  - 異なる表情: 5-6種類
- **前処理**: 画像リサイズ（512x768）、顔領域トリミング
- **キャプション作成**: 各画像に "a woman, professional outfit, neutral background" 等のテキストラベル付け

#### ステップ2: LoRA 学習実行
- **プラットフォーム**: Google Colab無料枠 (T4 16GB) or ローカルGPU
- **パラメータ**:
  - Learning Rate: 0.0001-0.0004
  - Iterations: 500-1000
  - Train Batch Size: 4-8
  - LoRA Rank: 32-64
- **コスト**: ¥0（Colab無料枠 or ローカル）
- **時間**: 5-15分/回

#### ステップ3: モデルマージ・テスト
- **マージ**: LoRA重み + ベースモデル（FLUX.2 [klein] 4B or FLUX.1 [schnell]）の統合
- **テスト生成**: 同じプロンプトで複数枚生成し、一貫性評価
- **目標**: 一貫性スコア 98% 以上（顔・髪型・服装が統一）

#### ステップ4: InstantID による顔補正（オプション）
- **用途**: 一貫性をさらに高める、複雑な背景での顔保持
- **手順**:
  1. テンプレート画像（基準顔）を選定
  2. 生成画像に InstantID 適用
  3. 顔パラメータ統一（strength = 0.9）

### 2-2. 一貫性チェックリスト
- [ ] 顔の輪郭が 95% 以上一致
- [ ] 目・眉・唇のパターンが統一
- [ ] 髪型・髪色が同一
- [ ] 肌トーンが統一
- [ ] 表情バリエーション 5-6種類で再現
- [ ] 照明・背景が任意に変更可能

### 2-3. 再学習サイクル
- **頻度**: 月1回（新しい衣装・背景追加時）
- **コスト**: ¥0（Colab / ローカル実行）
- **効果**: フレッシュさ維持、多様性確保

## 8. 法的・規約リスク対策

### 8-1. Instagram AI 関連規約

#### リスク: AI生成コンテンツの開示
- **Instagram規約**: "AIで生成または操作されたメディア" は `#aiinformation` ハッシュタグで開示が推奨
- **対応**:
  - [ ] すべての投稿に「このキャラクターはAI生成です」と明記
  - [ ] `#AIインフルエンサー` `#AI生成画像` タグを全投稿に付与
  - [ ] バイオに「This account features AI-generated content」と表示
- **ペナルティ**: 非開示の場合、アカウント停止の可能性

#### リスク: 著作権・人格権侵害
- **問題**: LoRA学習に使用する画像の権利
- **対応**:
  - [ ] LoRA学習用画像は **オリジナル撮影** or **商用利用可ライセンス** から入手
  - [ ] 有名人の顔を参考にしない（Deepfake扱いされる）
  - [ ] InstantID 使用時は同様に注意
- **ペナルティ**: DMCA申告 → 投稿削除 → アカウント警告

### 8-2. X（Twitter）AI 関連規約
- **リスク**: 「操作・改ざん」メディア表示
- **対応**:
  - [ ] メディア説明文に「AI-generated image」を記載
  - [ ] ツイートにコンテキスト警告ラベルが自動付与される可能性あり
- **ペナルティ**: 低クレーム（実際のペナルティは少ない）
- **⚠️ 重要**: X API は2026年2月7日から完全有料化。無料プランでの自動投稿は不可。手動投稿 or IFTTT連携に切替が必要

### 8-3. 広告・スポンサー関連規約

#### リスク: ステマ（ステルスマーケティング）
- **FTC/景表法**: スポンサードコンテンツは `#PR` `#ad` で開示必須
- **対応**:
  - [ ] アフィリエイト投稿には必ず `#ad` `#AD` を先頭に付与
  - [ ] スポンサー案件の特定条項を契約時に確認
  - [ ] インスタグラムの「Branded Content」ツール使用
- **ペナルティ**: 罰金 $5,000-$10,000 / アカウント停止

#### リスク: 金融商品勧奨（Mika 選択時）
- **金融商品取引法**: 未登録で有価証券の勧誘をすると違法
- **対応**:
  - [ ] 投稿に「これは個人の見解であり、投資アドバイスではありません」と免責文表記
  - [ ] 特定銘柄の強気推奨は避ける（教育目的に留める）
  - [ ] 「こういった企業もある」という紹介に留める
- **ペナルティ**: 警告書 → 行政処分 → 刑事罰（重度）

### 8-4. 個人情報保護規制（GDPR/CCPA）
- **リスク**: フォロワーデータの扱い
- **対応**:
  - [ ] プライバシーポリシーを HP に掲載
  - [ ] DM での個人情報収集は最小限
  - [ ] Google Analytics は GDPR 対応設定に
- **ペナルティ**: 罰金 €20M or 売上 4%（GDPR）
