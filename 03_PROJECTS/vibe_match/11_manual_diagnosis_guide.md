# 手動印象タグ診断ガイド

> DMで写真を受け取ったら、このガイドに従って診断し、結果を返す。
> Claude に写真の説明（特徴）を伝えて判定を依頼してもOK。

---

## 診断フロー（1人5分）

```
1. 写真を受け取る（DM）
2. 下記チェックシートで特徴を確認
3. 8カテゴリからTop3を判定
4. 結果テンプレートに記入
5. DMで返信
```

---

## チェックシート（写真を見ながら確認）

### A. 輪郭の印象
- [ ] 丸みがある（曲線的）→ ナチュラル/キュート/ウォーム寄り
- [ ] シャープ（直線的）→ クール/ワイルド/ミステリアス寄り
- [ ] 卵型（バランス）→ フレッシュ/エレガント寄り

### B. 目元の印象
- [ ] 大きくてくりっと → キュート/ウォーム
- [ ] 切れ長・涼しげ → クール/ミステリアス
- [ ] 穏やかで柔らかい → ナチュラル/ウォーム
- [ ] 力強い・鋭い → ワイルド/クール
- [ ] 透明感がある → フレッシュ
- [ ] 深みがある・吸い込まれる → ミステリアス/エレガント

### C. 全体の雰囲気（第一印象）
- [ ] 親しみやすい・安心する → ナチュラル/ウォーム
- [ ] 爽やか・清潔感 → フレッシュ
- [ ] 可愛い・守りたくなる → キュート
- [ ] 華やか・存在感 → エレガント
- [ ] 知的・スタイリッシュ → クール
- [ ] 力強い・アクティブ → ワイルド
- [ ] 惹きつけられる・謎めいた → ミステリアス
- [ ] 温かい・包み込むような → ウォーム

### D. パーツの濃さ
- [ ] あっさり（薄い）→ フレッシュ/ナチュラル（塩顔系）
- [ ] 普通（バランス）→ エレガント/ウォーム（しょうゆ顔系）
- [ ] はっきり（濃い）→ ワイルド/エレガント（ソース顔系）

---

## 判定マトリクス（迷った時の参考）

| 曲線的＋フレッシュ寄り | → **キュート** or **フレッシュ** |
| 曲線的＋成熟寄り | → **エレガント** or **ナチュラル** |
| 直線的＋フレッシュ寄り | → **フレッシュ** or **クール** |
| 直線的＋成熟寄り | → **クール** or **ミステリアス** |
| 力強い＋活動的 | → **ワイルド** |
| 穏やか＋温かい | → **ウォーム** |

## Claudeに判定を依頼する場合のプロンプト

```
以下の人物の写真の特徴から、印象タグTop3を判定してください。

【写真の特徴】
- 輪郭: （丸い/シャープ/卵型）
- 目元: （大きい/切れ長/穏やか/力強い）
- 雰囲気: （親しみやすい/爽やか/可愛い/華やか/知的/力強い/謎めいた/温かい）
- パーツの濃さ: （あっさり/普通/はっきり）
- その他の印象: （自由記述）

【8カテゴリ】
ナチュラル/フレッシュ/キュート/エレガント/クール/ワイルド/ミステリアス/ウォーム

Top3をパーセンテージ付きで判定し、それぞれ1行で理由も書いてください。
また、「統一感で合うタイプ」と「補完で合うタイプ」も1つずつ提案してください。
```
