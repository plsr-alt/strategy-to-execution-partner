## 前提
- **MVP期間**: 3ヶ月（2026-03 〜 2026-06）
- **開発体制**: 個人 or 少人数（1-2名想定）
- **技術スタック**: React Native (Expo) + Go + FastAPI + PostgreSQL + Redis + S3

## クリティカルパス

```
認証 → プロフィール → 写真アップ → AI診断エンジン → レコメンドエンジン → マッチング
```

**最もリスクが高いタスク**: 3.5（CLIP印象タグ推定エンジン構築）
- AI精度がプロダクト価値の核
- 早期にPoC検証すべき（Week 1-2でプロトタイプ推奨）

---

## 早期検証ポイント（Week 1-2に実施推奨）

1. **AI PoC**: CLIPで日本人の顔写真→印象タグ推定の精度確認
   - 5-10枚の写真で手動評価
   - タグのテキストプロンプト調整
2. **補完マッチの相性行列**: 少人数（5ペア）で「この組み合わせ良い？」を主観評価
3. **UIモック**: Figmaで主要5画面のモック → ユーザーインタビュー
