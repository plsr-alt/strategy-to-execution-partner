# VibeMatch 先行登録LP（Googleフォーム版）

> Googleフォームで作る。費用¥0。5分で完成。

---

## 作り方

1. https://forms.google.com にアクセス
2. 「新しいフォーム」作成
3. 以下の内容をコピペ

---

## フォーム内容

### タイトル
```
印象タグ × 雰囲気マッチング ― 先行登録
```

### 説明文
```
「条件は合うのに、なんかピンとこない」

マッチングアプリでそう感じたこと、ありませんか？

それは "雰囲気の相性" が合っていないからかもしれません。

私たちは、写真1枚から「印象タグ」をAIで診断し、
雰囲気の相性で相手を探せるマッチングアプリを開発中です。

✅ 似てる判定・外見採点は一切しません
✅ 「統一感」「補完」の2モードで探索
✅ なぜこの人が合うか、理由が見える

先行テスターとして参加してくれる方を募集しています。
```

### 質問1（必須）
```
メールアドレス
→ リリース時にご連絡します
```

### 質問2（任意・ラジオボタン）
```
あなたの年代を教えてください
- 18-24歳
- 25-29歳
- 30-34歳
- 35-39歳
- 40歳以上
```

### 質問3（任意・ラジオボタン）
```
マッチングアプリの利用経験は？
- 現在使っている
- 過去に使ったことがある
- 使ったことがない
```

### 質問4（任意・ラジオボタン）
```
どちらの探し方に興味がありますか？
- 統一感（似た雰囲気の人）
- 補完（対照的な雰囲気の人）
- 両方試したい
- よくわからない
```

### 質問5（任意・自由記述）
```
マッチングアプリで困っていることがあれば教えてください（任意）
```

### 送信後メッセージ
```
ご登録ありがとうございます！
リリース準備が整い次第、メールでご連絡します。

X（旧Twitter）でも最新情報を発信しています：
@恋愛改善設計士 ← あなたのアカウント名に差し替え
```

---

## フォームの設定

- 「メールアドレスを収集する」をON
- 「回答を1回に制限する」をON
- テーマカラー: 紫系（#6C5CE7）
