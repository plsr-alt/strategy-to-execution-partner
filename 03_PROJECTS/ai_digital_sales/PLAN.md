**作成日**: 2026-03-07
**プロジェクト名**: AI画像デジタル販売（BOOTH/pixivFANBOX/Gumroad複合戦略）

---

## 1. 背景・機会

### 1.1 現状
- AI画像生成インスタンス稼働中（FLUX.2、LoRA等）
- インフラ・生成エンジンは既存稼働状態（追加投資不要）
- TOEIC学習やYouTube案件と並行して実施可能

### 1.2 市場事例
- **pixivFANBOX**: 2ヶ月で40万円実績
  - 月額支援型（少量・高粘着性）
  - 国内オタクマーケット強固
- **BOOTH**: 3ヶ月で月10万円達成事例
  - 単品販売（流動的・拡散性高）
  - AI作品積極推進へシフト（2024年規約改定）
- **DLsite**: AI専用ブース設置済み（2024年2月）
  - 成人向けも合法化
  - 既存ユーザー規模大
- **先行事例「きゃんでぃタイム」**: BOOTH+Kindle+YouTube+FANZA複合で月10万円達成

### 1.3 なぜ今か？
1. **規制解禁**: AI作品販売が許可へシフト（2024年以降）
2. **市場拡大**: AI画像需要が急増（SNS用素材、ファンアート等）
3. **プラットフォーム競争**: pixiv/BOOTH/DLsite が AI商人積極獲得開始
4. **低リスク**: 初期投資最小、既存インフラ活用

## 2. 事業コンセプト

### 2.1 コア戦略
**複合チャネル戦略**で単一チャネルのリスク回避：
1. **pixivFANBOX** → 月額支援（ファン固定化）
2. **BOOTH** → 単品販売（流動的売上）
3. **DLsite** → 大規模流通（スケール）
4. **Gumroad** → 海外展開（USD稼ぎ）

各チャネルで**同じコンテンツを異なる価格帯・形式で再販売** → 効率最大化

### 2.2 コンテンツジャンル（複数で分散リスク）
1. **AIグラビア/コスプレ写真集**（メイン・高単価）
   - グラビア好きなコミュニティが購買力高
   - FANBOX 月額支援と相性良好

2. **AIイラスト素材パック**（拡大需要・低リスク）
   - SNS用アイコン、バナー等
   - 初心者デザイナー購買層

3. **AIアート壁紙・ポスター**（中価格帯）
   - 繰り返し購買層（季節ごと）
   - BOOTHで売上性高

4. **AIキャラクター設定画集**（高付加価値・長期販売）
   - クリエイター層向け
   - リピート率高い

## 4. 実現メカニズム

### 4.1 なぜ達成可能か

**条件1: コンテンツ量**
- 月間 5-8セット制作（自社AI生成で低コスト）
- 6ヶ月で 50+ セット蓄積
- ストック型ビジネス（過去コンテンツも売り続ける）

**条件2: プロモーション効率**
- pixiv 無料投稿で露出（SEO/バズ）
- → FANBOX リンク誘導（転換率 2-5%）
- → SNS シェア（X/note で拡散）
- 広告費 0円で実現

**条件3: 単価帯**
- グラビア: 500-1,200円（高粘着・高単価）
- 素材: 300-500円（低価格・回転率高）
- 複合販売で平均単価 600-800円維持

**条件4: チャネル分散**
- 1つのコンテンツを 4-5 チャネルで販売
- リスク分散（1プラットフォーム規制→他でカバー）

### 4.2 損益分岐点

```
初期投資: 21,500円（ソフト購入等）
月次コスト: 200-3,000円（最小化）

月2で 25,000円売上 → 3ヶ月で初期投資回収
月3以降は全てが利益
```

## 5. リスク要因

### 5.1 主要リスク

| リスク | 発生確率 | 対策 |
|---|---|---|
| **プラットフォーム規約変更** | 中 | 複数チャネル展開で依存回避 |
| **AI規制強化** | 低-中 | 透明性重視（AI表記徹底） |
| **競合増加** | 高 | 品質＆スピード差別化 |
| **規約違反で削除** | 低 | 規約遵守体制（チェックリスト） |

### 5.2 対応策

1. **チャネル多角化**: pixiv 依存を避け 4つ以上で展開
2. **AI表記透明性**: 「AI生成」を全商品に明記（信頼構築）
3. **品質維持**: AI生成品質を継続改善（LoRA学習）
4. **コミュニティ**: FANBOX 支援者とのコミュニティ化（離脱抑止）
