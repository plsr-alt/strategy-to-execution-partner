# AI画像デジタル販売仕様書（SPEC.md）

## 1. プロジェクト概要

### 1.1 ビジネスモデル
複数のデジタル販売プラットフォームを組み合わせたAI生成画像コンテンツの販売事業。
- **現状**: FLUX.2、LoRA等のAI画像生成インスタンス稼働中
- **目標（6ヶ月）**: 月間売上10～20万円達成
- **KPI**:
  - 月1: 0～1万円（初期構築フェーズ）
  - 月3: 3～5万円（ファン基盤形成）
  - 月6: 10～20万円（複数チャネル展開）

## 3. コンテンツ制作ワークフロー

### 3.1 コンテンツカテゴリ定義

| カテゴリ | 説明 | 制作工数 | 単価目安 | AI最適性 |
|---|---|---|---|---|
| **AIグラビア/コスプレ写真集** | AI生成ハイクオリティ写真集 | 1セット=2-3日 | FANBOX:1,000円, BOOTH:500-1,000円 | ⭐⭐⭐⭐⭐ |
| **AIイラスト素材パック** | SNS用アイコン、バナー、テンプレート | 1パック=1日 | BOOTH:300-800円 | ⭐⭐⭐⭐ |
| **AIアート壁紙・ポスター** | 高解像度壁紙（2K/4K）、ポスターデータPDF | 1セット=4-6時間 | BOOTH:500-1,200円 | ⭐⭐⭐⭐⭐ |
| **AIキャラクター設定画集** | キャラ設定集（複数角度、衣装違い） | 1キャラ=3-4日 | BOOTH:1,000-3,000円 | ⭐⭐⭐ |
| **AIアニメーション素材** | 短編ループアニメーション（GIF/MP4） | 1本=2-3日 | FANBOX:追加オプション | ⭐⭐⭐ |

### 3.2 制作ワークフロー（標準フロー）

```
Step 1: コンセプト・スタイル決定（30分）
  └→ 「AI美女グラビア」「和風イラスト素材」など方向性決定

Step 2: プロンプト設計 + AI生成（2-4時間）
  └→ FLUX.2 [klein] 4B + LoRA（ローカル or Google Colab無料枠）
  └→ 品質チェック（50-100枚の中から10-20枚厳選）

Step 3: 画像後処理・加工（1-2時間）
  └→ 構図確認、色調補正、不自然部分修正
  └→ GIMP / Canva無料版 で微調整

Step 4: パッケージング（1-2時間）
  └→ サムネイル作成、説明文作成、タグ付け
  └→ メタデータ埋め込み

Step 5: アップロード + 販促（1時間/PF）
  └→ 各プラットフォームに投稿
  └→ SNS告知（X/pixiv/note等）

合計工数: 1セット 5-10時間 → 単価平均800円 = 80-160円/時間（効率化で改善）
```

### 3.3 品質基準

| 項目 | MVP基準 | 理由 |
|---|---|---|
| **解像度** | 最小 1024×1024px（推奨 2048×2048以上） | プラットフォーム要件 + ユーザー期待 |
| **AI生成品質** | 顔・手に破綻なし、構図明確 | グレード感を維持 |
| **統一性** | スタイル・配色シリーズ化（3セット以上） | ユーザー「推し」形成 |
| **著作権対応** | プロンプトに有名キャラなし。オリジナル設定 | BOOTH/pixiv規約対応 |
| **AI表記** | 全販売ページに「AI画像」と明記 | 規約要件 + ユーザー信頼構築 |

## 5. プロモーション戦略


## 6. 法的要件・コンプライアンス

### 6.1 AI画像表記義務

| プラットフォーム | 表記義務 | 表示場所 | ペナルティ |
|---|---|---|---|
| **pixivFANBOX** | 推奨（必須ではない） | 説明欄「AI生成」タグ | なし（フォロワー信頼喪失） |
| **BOOTH** | **必須** | 商品タイトル/説明欄 | 削除・垢BAN（規約改定進行中） |
| **DLsite** | 必須（カテゴリ選択） | AI専用ブース登録 | 別カテゴリ登録禁止 |
| **Gumroad** | 推奨 | 説明欄 | 信用低下 |

**MVP対応**: 全プラットフォームで「AI生成」を明記。透明性優先。

### 6.2 成人向けコンテンツ規約

| PF | グラビア（局部なし） | グラビア（局部表現） | 規制強度 |
|---|---|---|---|
| pixivFANBOX | ✅ OK | ⚠️ 要確認（R-15推奨） | 中 |
| BOOTH | ✅ OK | ⚠️ 年齢確認必須 | 弱 |
| DLsite | ✅ OK | ✅ OK（成人タグ） | 強 |
| Gumroad | ✅ OK | ✅ OK（条件あり） | 中 |

**MVP方針**: 局部なし「セクシー系グラビア」で統一（規制回避 + 海外対応性高）

### 6.3 著作権・肖像権対応

| 確認項目 | MVP対応 | 理由 |
|---|---|---|
| **プロンプトにキャラ指定なし** | ✅ | 版権キャラは即規約違反 |
| **人物は全てAI生成または自作モデル** | ✅ | 実在人物の肖像権回避 |
| **生成画像の再利用禁止** | ✅ | 規約に明記（コンテンツ保護） |
| **素材ライセンス表記** | ✅ | 使用素材（背景等）のクレジット記載 |

## 7. コスト・採算試算

### 7.1 初期投資（Phase 0）

| 項目 | コスト | 理由 |
|---|---|---|
| **AI生成インスタンス（FLUX.2推論サーバ）** | 0円 | 既存稼働（追加投資不要） |
| **画像編集ソフト（GIMP / Canva無料版）** | 0円 | OSSまたは無料枠で十分 |
| **ドメイン + Webサイト（ポートフォリオ）** | 1,500円/年 | pixiv プロフ + Twitter で代替可 |
| **アカウント開設・手数料** | 0円 | 無料 |
| **初期コンテンツ制作（50セット分）** | 0円 | 内製（時給換算なし） |
| **小計** | **1,500円** | ドメインのみ（不要なら¥0） |

### 7.2 月次ランニングコスト（Phase 1-6）

| 項目 | 月額 | 年額 | 備考 |
|---|---|---|---|
| **クラウドストレージ（OneDrive/Google Drive）** | 200円 | 2,400円 | 100GB程度で十分 |
| **AI生成（FLUX.2 [klein] ローカル）** | 0円 | 0円 | Apache 2.0, ローカル実行 |
| **プロモーション費（広告なし、SNS発信のみ）** | 0円 | 0円 | オーガニック展開 |
| **プラットフォーム手数料（売上から天引き）** | 売上×5-10% | - | BOOTH 5.6% / FANBOX 10% |
| **小計** | **200円** | 2,400円 | ストレージのみ |

### 7.3 損益分岐点分析（6ヶ月目）

```
初期投資: 1,500円（ドメインのみ。不要なら¥0）
月次コスト（平均）: 200円（クラウドストレージのみ）

月6 売上予測: 154,000円
  ├→ 手数料（平均7%）: -10,780円
  ├→ ランニングコスト: -200円
  └→ 営業利益: 143,020円/月

ROI（初期投資）: 1,500円 ÷ 143,020円 = 1% / 月
→ 初月で初期投資回収

累積利益（6ヶ月）:
  売上 422,000円 - 手数料 29,540円 - 初期投資 1,500円 - 運用コスト 1,200円
  = 389,760円（純利益）
```

**結論**: 初期投資ほぼゼロ。全ツール無料化により初月から黒字。複利効果で加速可能。
