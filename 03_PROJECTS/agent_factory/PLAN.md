# Agent Factory — PLAN

## やりたいこと
- 公式ドキュメントを唯一の事実源として、AIエージェントシステムを自動生成する"工場"を構築
- 「書くAI」と「疑うAI」を分離して、Hallucination を構造的に防ぐ
- ドメインを入力するだけでエージェント一式が出てくる再現可能なパイプライン

## きっかけ
- Claude Code公式ドキュメント27ページの情報が多く矛盾もあり全体像が掴みにくい
- → 事実源を一元化し、段階的に圧縮・生成する仕組みが必要

## 最小実装の方針
1. sources/ に公式ドキュメントを格納（唯一の事実）
2. writer/checker 分離パイプラインで設計ガイドを生成
3. 設計ガイドを圧縮してdigestに
4. digestを埋め込んだジェネレーターでドメイン特化エージェントを生成
5. 重い処理はWSL+Groqに委譲（Claudeトークン節約）
