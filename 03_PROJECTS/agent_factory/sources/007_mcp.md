# Model Context Protocol (MCP)
Source: https://code.claude.com/docs/en/mcp

## What is MCP
Open source standard for AI-tool integrations. Connects Claude Code to external tools, databases, APIs.

## Install MCP Servers

### Option 1: Remote HTTP (recommended for cloud services)
```bash
claude mcp add --transport http <name> <url>
# Example: claude mcp add --transport http notion https://mcp.notion.com/mcp
# With auth: claude mcp add --transport http secure-api https://api.example.com/mcp --header "Authorization: Bearer token"
```

### Option 2: Remote SSE (deprecated, use HTTP instead)
```bash
claude mcp add --transport sse <name> <url>
```

### Option 3: Local stdio (for local processes, system access)
```bash
claude mcp add [options] <name> -- <command> [args...]
# All options BEFORE server name; -- separates from command
# Example: claude mcp add --transport stdio --env AIRTABLE_API_KEY=KEY airtable -- npx -y airtable-mcp-server
```

## Manage MCP
```bash
claude mcp list
claude mcp get github
claude mcp remove github
/mcp  # within Claude Code: check status, authenticate
```

## MCP Scopes
| Scope | Location | Use |
|-------|----------|-----|
| local (default) | ~/.claude.json under project path | Personal, current project only |
| project | .mcp.json in project root (committed) | Team-shared |
| user | ~/.claude.json | Personal, all projects |

## .mcp.json Format
```json
{
  "mcpServers": {
    "shared-server": {
      "command": "/path/to/server",
      "args": [],
      "env": {"KEY": "${ENV_VAR:-default}"}
    }
  }
}
```
Supports: ${VAR} and ${VAR:-default} environment variable expansion in command, args, env, url, headers

## Scope Hierarchy
local > project > user (same name → highest priority wins)

## OAuth 2.0 Authentication
- Use /mcp → "Authenticate" in Claude Code
- Tokens stored securely, refreshed automatically
- --callback-port: fix port for pre-registered redirect URIs
- --client-id + --client-secret: pre-configured credentials

## MCP Output Limits
- Warning threshold: 10,000 tokens
- Default max: 25,000 tokens
- Override: MAX_MCP_OUTPUT_TOKENS env variable

## MCP Resources
- Reference: @server:protocol://resource/path
- Type @ to see available resources in autocomplete

## MCP Tool Search
- Auto-enables when MCP tool definitions exceed 10% of context
- ENABLE_TOOL_SEARCH=auto (default) / auto:N% / true / false
- Requires Sonnet 4+ or Opus 4+

## Claude as MCP Server
```bash
claude mcp serve  # Start Claude Code as stdio MCP server
```

## Managed MCP
- managed-mcp.json: exclusive control (users cannot add servers)
  - Locations: /Library/Application Support/ClaudeCode/managed-mcp.json (macOS)
- allowedMcpServers + deniedMcpServers in managed settings: policy-based control
