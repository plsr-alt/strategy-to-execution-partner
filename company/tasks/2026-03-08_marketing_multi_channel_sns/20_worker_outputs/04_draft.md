# マルチチャネルSNS自動展開システム — マーケティングチーム設計書

**バージョン**: v1.0
**作成日**: 2026-03-08
**対象期間**: 2026年3月〜5月（Phase 1-3）
**予算**: ¥0/月（EC2既存共用）
**運用体制**: 1人運用 + AI自動化

---

## エグゼクティブサマリー

### ビジョン
YouTube既存資産（月産15本動画）を3つのSNS（Instagram / Pinterest / X）に自動展開し、月産45本以上の投稿を実現。¥0の予算制約下で、認知拡大から月500-1000の訪問流入、月+50のYouTube登録者獲得、月$100-200の収益化を実現する完全自動化フレームワーク。

### 勝ち筋
- **差別化**: AI開示を優位性に転換（「AIがAIを教える」）
- **効率化**: 1本のYouTube動画から3つのプラットフォーム用テキスト・画像を自動生成・投稿
- **スケーラビリティ**: EC2既存リソースで¥0運用、拡張可能な非同期アーキテクチャ

### 成功要因
- テキスト変換（Groq）とスケジューリング（APScheduler）による完全自動化
- プラットフォーム別の訴求最適化（ビジュアル / SEO / リアルタイム）
- 週次のA/Bテストと月次のKPI検証による継続改善体制

## Section 1: 全体設計 — YouTube横展開の思想・勝ち筋

### 1.1 基本戦略

YouTube月産15本の動画は、ブランドの最大資産。これを以下の3つの視点で展開：

| 視点 | Instagram | Pinterest | X |
|------|-----------|----------|---|
| **主軸** | ビジュアル・即座な認知 | 検索SEO・情報資産化 | リアルタイム・会話 |
| **ユーザー属性** | 25-45歳、ビジュアル重視 | 計画購買層、女性比率高い | テック業界、金融層 |
| **リーチ寿命** | 数日〜数週間 | 3-6ヶ月（長期参照） | 18分（短期） |
| **投稿形式** | Carousel（複数画） + ハッシュタグ | 標準ピン + SEOキーワード | テキスト280字以下 + 手動投稿 |
| **期待効果** | 月+500-1000訪問 | 月+300-500訪問 + 資料DL | 月+100-200訪問 + リアルタイム議論 |

### 1.2 ビジネスモデル

```
YouTube動画（月15本）
      ↓
   ├─→ Instagram 月15投稿（3時差投稿 = 月45本）
   ├─→ Pinterest 月15投稿（初回+再ピン = 月30本相当効果）
   └─→ X 月15投稿（手動 = 月15本）
      ↓
 月+500-1000訪問流入
      ↓
 YouTube登録者増（月+50）→ 繰り返し視聴
      ↓
 月$100-200収益化（AdSense + ガジェットアフィリエイト）
```

|------|------|
