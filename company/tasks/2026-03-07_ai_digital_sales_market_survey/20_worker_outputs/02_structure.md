## フレームワーク: プラットフォーム 3C分析

### 1. BOOTH
**Customer**: カジュアルから本格クリエイター（広い層）
**Competitor**: Gumroad、FANBOX（定期）、DLsite
**Company**: pixiv系、低手数料(5.6%+45円)、参入障壁最低

**強み**:
- 手数料最安
- 誰でも参入可
- 単品販売に特化

**弱み**:
- FANBOX との食い合い
- ブランド認知度では FANBOX < DLsite
- SNS連携機能がやや弱い

---

### 2. pixivFANBOX
**Customer**: 推し活ファン、クリエイター応援層（濃い）
**Competitor**: YouTube Member、Patreon
**Company**: pixiv系、pixiv 1,350万ユーザーの基盤、10%+180円手数料

**強み**:
- pixiv ユーザー基盤が強力（1,350万人）
- ファンコミュニティ構築向き
- 定期支援で安定収入

**弱み**:
- 手数料が高い（10%+180円）
- BOOTH との分散必要
- 初期ファン集めの難しさ

---

### 3. DLsite
**Customer**: ニッチ・18禁購買層（深い）
**Competitor**: BOOTH（AI可）、メロンブックス
**Company**: ビジュアルノベル・ゲーム販売の老舗、高手数料(30～50%)

**強み**:
- 18 禁 / アダルル分野で認知度高い
- セット販売・大型コンテンツ向き
- 購買意欲の高い層が集中

**弱み**:
- 手数料が高い（30～50%）
- 18禁取扱いのハードル
- 年齢層がやや高め

## 価格帯最適化（プラットフォーム別）

### BOOTH（単品売切型）
```
入門層（初心者）:  300～500円 / 画像1～3枚
標準層:         500～1,000円 / 画像3～10枚
上級層（セット）:  1,500～3,000円 / 画像20～50枚 + 高解像度
```

推奨: **500円 × 10個SKU** > **2,000円 × 1個SKU** の複数SKU戦略

### pixivFANBOX（定期支援型）
```
エントリー:   500円/月（限定画像3～5枚）
スタンダード: 1,000円/月（早期公開・高解像度・カスタムリクエスト5回/月）
プレミアム:   3,000円/月（月5～10枚、DM相談権、グッズ相談権）
```

推奨: **3段階併置** で月額10～30万円の安定収入

### DLsite（セット販売型）
```
小型パッケージ:   500～1,000円 / 10～20画像
標準パッケージ: 1,500～2,000円 / 50～100画像
大型パッケージ: 3,000～5,000円 / 200+画像 + 素材セット
```

推奨: **月2回以上の更新** で常時ランキング上位維持

## リスク分析

| リスク | 発生確度 | 影響度 | 対策 |
|--------|--------|--------|------|
| **著作権侵害クレーム** | 中 | 高 | オリジナル or 十分なアレンジ、利用規約確認 |
| **プラットフォーム規約変更** | 中 | 高 | 複数プラットフォーム並立で分散 |
| **AI申告義務違反** | 中 | 高 | 購入者欄・説明に明記「AI生成」 |
| **模倣・盗用** | 低～中 | 中 | 水印・シリアル・限定販売 |
| **市場飽和（イラスト分野）** | 高 | 中 | ニッチ特化・キャラ一貫性で差別化 |
| **手数料率引き上げ** | 低 | 中 | 複数プラットフォーム分散 |
