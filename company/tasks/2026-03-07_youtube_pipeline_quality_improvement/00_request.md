# タスク依頼: YouTube自動化パイプラインの全面品質改善

## 依頼日
2026-03-07

## 依頼者
ユーザー（事業責任者）

## 依頼内容
YouTube自動化パイプラインの全面品質改善。ユーザーフィードバックに基づき、競合調査→戦略策定→実装計画を立てる。

## ユーザーフィードバック（全項目対応必須）

1. **音声が遅い・AI臭い**: gTTSの読み上げが不自然。よく再生されているチャンネルを調査し、自然なスピード・トーンに合わせる
2. **映像と内容がマッチしていない**: ただの風景動画になっている。説明テキスト・図解・イラストを使ってわかりやすくする。100K再生以上の主要動画を確認して参考にする
3. **動画の長さがまちまち**: 15分や8分とバラバラ。最低20分以上に統一
4. **サムネイルができていない**: 現状のPIL生成サムネが機能していない
5. **チャンネルアイコンが初期値**: ブランディングされていない
6. **チャンネル名がデフォルト**: 始めたてがバレバレ

## 調査要件
- YouTube で「お金 投資 サラリーマン」「節約 貯金」「資産運用」系の100K再生以上の日本語動画を分析
- 音声（スピード、トーン、間の取り方）
- 映像構成（テロップ、図解、アニメーション、背景）
- サムネイルデザイン（文字数、配色、レイアウト）
- チャンネルブランディング（名前、アイコン、バナー）
- 動画の長さ・構成パターン

## 期待する成果物
1. 競合分析レポート（具体的なチャンネル名・動画URL・特徴）
2. youtube_pipeline.py の改善仕様書（具体的なコード変更箇所）
3. チャンネルブランディング提案（名前・アイコン・バナーデザイン案）
4. 最初の3本の動画テーマ案

## 技術的制約

- YouTube Data API v3 でアップロード
- 予算制約: 無料〜低コストAPIのみ（ElevenLabsは申請中だがまだ未取得）
