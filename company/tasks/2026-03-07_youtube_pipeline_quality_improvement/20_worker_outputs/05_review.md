#### 5. C品質チェック — infographic keywords抽出ロジック不明瞭
**現状**: 成果物2 L331「keywords = theme.get("keywords", ["ポイント1", "ポイント2", "ポイント3"])」で、generate_theme()から正しく渡されるか不明確

**修正案**:
成果物2に以下を追記:
```markdown
## テーマデータスキーマ（generate_theme() → edit_video()への引き渡し）

theme: dict = {
    "theme": str,           # 動画タイトル（例: "ChatGPT×家計簿で月5万円節約する完全ガイド"）
    "hook": str,            # フック文（例: "月5万円節約、ChatGPTで本当にできます"）
    "keywords": list[str],  # キーワード3-4個（例: ["ChatGPT", "家計管理", "節約", "AI"]）
    "sections": list[dict], # 5セクションデータ
        - {
            "title": str,   # セクションタイトル
            "content": str, # セクション本文（400-1200文字）
          }
}

# 動作確認: theme["keywords"] が list型かつ length >= 3 であることをassert
assert isinstance(theme["keywords"], list) and len(theme["keywords"]) >= 3, \
    "テーマに keywords (list, 3以上) が含まれていません"
```
