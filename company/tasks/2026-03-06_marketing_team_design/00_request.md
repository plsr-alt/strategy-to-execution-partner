## ブランド概要
- **名前**: 恋愛改善設計士
- **タグライン**: 恋愛は、設計できる。
- **ターゲット**: 30-40代男性。仕事はできるのに恋愛だけ詰まっている人。論理的思考ができる。煽り系に嫌悪感。「モテたい」より「ちゃんとした関係を築きたい」
- **差別化**: 煽らない・女性否定しない。構造的・論理的アプローチ。「改善」訴求
- **商品**: 無料note記事 → セルフ診断キット（無料） → 有料note（500円） → 個別セッション（10,000円）
- **ファネル**: X投稿 → プロフィール → 固定ポスト → note → 診断 → DM → セッション

## 制約条件
- 運営体制: 1人（本業あり・副業）
- 可処分時間: 週5-7時間（平日夜30-60分 + 隙間時間）
- 月予算: ほぼ0円（初期段階、Tier2ツールで月$5-22程度まで検討可）
- 顔出し: 不可
- 得意な形式: 長文テキスト、構造的コンテンツ、AIツール活用
- 避けたい運用: 毎日投稿の負担、煽り・炎上商法、ナンパ術系の匂い

## 使用SNS・メディア
現在: X、note
計画中: YouTube（金融・投資チャンネル6つ、自動化パイプライン構築済み）
検討可: Threads、Instagram、TikTok、Podcast、メルマガ
ツール: xrs (X Research Skills)、CrewAI、Groq、Claude Code

## 市場における強み・弱み
**強み**:
- AIツール・自動化に非常に強い
- 構造的・論理的アプローチは恋愛市場では希少
- 煽らないポジショニングは30-40代論理型男性に刺さる
- 低コスト運営

**弱み**:
- フォロワーゼロからのスタート
- 可処分時間が限られる
- 顔出し不可
- 実績・事例がまだない
