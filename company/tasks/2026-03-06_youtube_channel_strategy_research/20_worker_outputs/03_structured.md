# ジャンル×形式 スコアリングマトリクス — AI自動化YouTube戦略の構造化

## 全体構成

AI自動化YouTube市場における**ジャンル×形式の組み合わせ**を、6つの評価軸で定量的にスコアリング。優先度S/A/B/Cを決定し、最適ポートフォリオを提案する。

---

## スコアリング軸の定義（各1-5点）

### 評価軸の説明

| # | 軸名 | 定義 | スコアリング基準 |
|---|------|------|-----------------|
| **1** | **CPM（広告単価）** | 1,000再生時の期待広告収入（円）の相対評価 | 5点: 3,000円以上 / 4点: 1,500～3,000円 / 3点: 500～1,500円 / 2点: 100～500円 / 1点: 100円未満 |
| **2** | **収益化期間** | チャンネル収益化条件達成までの期間短さ（登録者1,000人＋過去12ヶ月4,000時間再生時間） | 5点: 3ヶ月以内 / 4点: 3～6ヶ月 / 3点: 6～12ヶ月 / 2点: 12～18ヶ月 / 1点: 18ヶ月以上 |
| **3** | **自動化適性** | 企画～投稿まで自動化可能な割合（%を5段階に変換） | 5点: 90～100% / 4点: 70～90% / 3点: 50～70% / 2点: 30～50% / 1点: 30%未満 |
| **4** | **BANリスク低さ** | 著作権侵害・AI生成コンテンツ規制での収益化停止リスクの低さ | 5点: 極低（CC0素材のみ対応可） / 4点: 低（人間工程で対応） / 3点: 中（著作権チェック要） / 2点: 高（複数リスク要素） / 1点: 極高（基本的に高リスク） |
| **5** | **競合飽和度低さ** | 新規参入時の競争環境の緩さ（飽和度が低いほど高スコア） | 5点: 極低飽和（参入障壁高い） / 4点: 低飽和 / 3点: 中飽和 / 2点: 高飽和 / 1点: 極高飽和 |
| **6** | **英語展開性** | 日本語版をそのまま英語化した場合のCPM倍率＆市場規模 | 5点: CPM 3倍以上（アメリカ・オーストラリア等） / 4点: CPM 2～3倍 / 3点: CPM 1.5～2倍 / 2点: CPM 1～1.5倍 / 1点: CPM 1倍未満（地域限定） |

## 上位候補（優先度S&A）の詳細分析

### S-1: 金融・投資・節約（長尺15～20分）

#### 基本指標
- **総合スコア**: 4.25 / 5.0
- **CPM**: 1,500～3,000円（日本市場最高水準）
- **月収期待値**: 30～50万円（月10～20本投稿時）
- **英語展開CPM**: $8～12（CPM 1,200～1,800円。日本の0.75倍だが、視聴数＆広告主で補完）

#### 自動化パイプライン
1. **企画・分析（人間）**: 投資トレンド、金融ニュース、節約テクニック（30分/本）
2. **台本作成（AI補助）**: ChatGPT で構成化、AI が段落生成（20分/本）
3. **素材調達（自動）**: チャート・グラフ生成（Python/Canva API）、統計データAPI連携
4. **ナレーション（AI）**: VoiceVox高品質音声（自動生成）
5. **編集・アップロード（自動）**: FFmpeg + yt-dlp スクリプト化

#### リスク対策
- ✅ **BANリスク**: 低（金融データは著作権保護なし）
- ⚠️ **規制リスク**: 中（「人間の分析・視点」を30%以上組み込む必須）
- ⚠️ **著作権**: 中（金融ニュースソース引用は出典明記推奨）

#### 推奨実装
```
自動化度: 50% （データ収集→編集は自動、企画・分析は人間）
月投稿本数: 10～20本（無理のないペース）
パイプライン構築期間: 2～3ヶ月
```

**根拠**: 01_research.md セクション 2.1, 7.2

### S-2: ゆっくり解説（科学・歴史・雑学、中尺8～12分）

#### 基本指標
- **総合スコア**: 3.86 / 5.0
- **CPM**: 100～300円（低単価だが完全自動化可能）
- **月収期待値**: 3～5万円（月30～50本投稿時）
- **英語展開**: 日本語音声は海外で不人気（スコア3点）。英語ナレーション版で1.5～2倍CPM化

#### 自動化パイプライン
1. **テーマ選定・台本（完全自動化）**: Wikipedia + GPT で科学トピック生成（5分/本）
2. **音声（自動化）**: Voiceroid（ずんだもん等）で朗読（3分/本）
3. **映像素材（自動化）**: CC0素材（Unsplash, Pexels）から自動ダウンロード・編集
4. **編集（完全自動化）**: テンプレート化（背景+字幕+転換効果）
5. **アップロード（自動）**: YouTube API で定時投稿

#### リスク対策
- ✅ **BANリスク**: 極低（CC0素材＆オリジナル台本のため著作権問題ゼロ）
- ✅ **規制リスク**: 低（Voiceroid音声は「生成AIです」チェック＋テロップで対応）
- ✅ **著作権**: CC0素材のみ、トレーサビリティ完全

#### 推奨実装
```
自動化度: 90%（ほぼフル自動化）
月投稿本数: 30～50本（毎日投稿可能）
パイプライン構築期間: 1ヶ月（テンプレート設計のみ）
初期投資: 低（Voiceroid/Canva Pro 月 1,500円程度）
```

**重要**: CPM低いため、「完全自動化＆大量投稿」で月3～5万円の安定収入を確保。他ジャンルのテスト用として最適。

**根拠**: 01_research.md セクション 2.2, 7.1

### S-3: ホラー・怪談朗読（オリジナル台本、中尺8～12分）

#### 基本指標
- **総合スコア**: 3.71 / 5.0
- **CPM**: 1,000～1,500円（高単価、金融系と競争可能）
- **月収期待値**: 30～50万円（月10～20本投稿時）
- **[推測] 穴場度**: ⭐⭐⭐⭐⭐（高CPM × 低競合 × 低著作権リスク）

#### 自動化パイプライン
1. **台本制作（人間またはAI+人間）**: オリジナル怪談創作（40分/本） または AI 生成台本を人間が修正（20分/本）
2. **著作権確認（人間）**: 既知作品との重複チェック（10分/本）
3. **ナレーション（AI自動）**: VoiceVox高品質音声で朗読
4. **映像素材（自動）**: 怖いフリー画像（Pixabay等）を自動検索・編集
5. **編集・アップロード（自動）**: テンプレート化

#### リスク対策
- ⚠️ **BANリスク**: 中（著作権3回警告で停止。オリジナル台本のみ使用で対応）
- ✅ **規制リスク**: 低（オリジナルコンテンツは「人間創作」判定）
- 🔴 **著作権**: **極高**（既知作品・2chまとめ朗読は著作権侵害。厳禁）

#### 推奨実装
```
自動化度: 70% （台本制作は人間工程、朗読・編集は自動）
月投稿本数: 10～20本
パイプライン構築期間: 1～2ヶ月
台本仕入れ源: オリジナル創作 / 著作権フリー素材サイト / AI生成後の人間修正
著作権確認フロー: 投稿前に3段階チェック（タイトル検索、内容チェック、出版社ガイドライン確認）
```

**根拠**: 01_research.md セクション 2.7, 3.2, 7.2, 5.2

### A-5: ランキング動画（TOP10等、中尺8分）

#### 基本指標
- **総合スコア**: 3.14 / 5.0
- **CPM**: 100～300円（低単価。量産で補完）

#### リスク対策

**[推測] 注意**: CPM低いため、月40～50本投稿の継続が必須。自動化が強みだが、「人間の工夫」をないがしろにすると収益化停止リスク。

### フレームワーク適用: SWOT分析（AI自動化YouTube事業）

```
STRENGTHS（強み）
├─ 自動化パイプライン構築能力（60～90%自動化可能）
├─ AI音声・画像生成での低コスト運用
├─ 複数ジャンル並列展開でリスク分散可能
└─ 英語展開で CPM 2～3倍、視聴数3倍以上
```
