## 要点（3〜5個）

1. **プラットフォーム別アルゴリズム特性**:
   - YouTube Shorts は「発見」重視（推奨度順ラッセル）で、完全視聴率が最優先指標
   - Instagram Reels は「保存」と「シェア」を重視（フィード内での持続性判定）
   - TikTok は「完全視聴率 + エンゲージメント」の加重平均で、短尺（15-30秒）が有利
   - **攻略法**: Shorts で長さ試験 (45-60秒), Reels で「保存促す」CTA, TikTok で高速カットを優先

2. **AI生成ショート動画の市場機会**:
   - 2023-2026年 CAGR 45-60% で成長中 (B2B/B2Cの両方で需要)
   - 特に金融・教育・エンタメで高需要（トレンド図解、スキル習得系)
   - AI生成コンテンツの収益化許可が 2024-2025年から段階的に許可開始 → AI Shorts Factory のタイミングは最適

3. **¥0予算1人運用の成功構図**:
   - FLUX.2 [klein] (ローカル無料) + Ken Burns (FFmpeg無料) + Pixabay/YT Audio (無料) で完全ゼロコスト実現可能
   - 月90本投稿 (3本/日) で月¥5-15万の推定広告収入達成ラインは業界標準（Creator Fund 条件達成後）
   - EC2共用 + GCS自動削除で月額コスト ¥0 実現（既存YouTube パイプラインとの共有）

4. **規約リスク最小化 → 主要対策**:
   - **AI顔生成禁止**: FLUX.2 [klein] 標準設定で人物生成低確率だが、プロンプトに「人物」含めない運用ルール化
   - **著作権**: Pixabay / YT Audio Library 限定で完全安全（Content ID マッチなし）
   - **ディスクロージャー**: キャプション最後に「#AIアート」タグで予防的明記
   - **金融免責**: 金融テーマは「投資助言ではありません」を必ず表記
   - **重複投稿**: 各PF内での同一コンテンツ重複 NG (アルゴリズム評価低下), 別PF間は OK

5. **競合チャネル分析と差別化ポイント**:
   - [推測] 現在 YouTube/TikTok/Instagram 上でAIショート動画専門チャネルは「数百～数千」存在（急増中）
   - 既存競合は「単一テンプレ駆動」（例: AI画像列挙型）で低粘着度 (avg. CTR < 1%)
   - **AI Shorts Factory の差別化**:
     - テンプレ多様化 (4種以上) で視聴者飽き防止
     - 金融 × AIアート の「複合テーマ」で検索ニッチ化
     - Ken Burns アニメーション で「静止画 → 動画」感を活かした高品質化
     - 3本/日投稿で「頻度信号」を強化 → アルゴリズム有利

## AI生成ショート動画市場 (2025-2026年)

### 市場規模・成長率
- **市場規模**: [推測] グローバル $2-5B (2025年), 日本国内 $50-200M
- **成長率**: CAGR 45-60% (2023-2028予測)
- **主な成長要因**:
  1. FLUX.2 [klein], Wan 2.1 等 OSS モデルの無料化
  2. YouTube / TikTok の API 無料枠拡大
  3. Creator Fund の段階的 AI コンテンツ許可
  4. コンテンツ生成速度上昇 (手作業 → 自動化)

### プレイヤー類型
| 類型 | 特徴 | 代表例 | リスク |
|------|------|--------|--------|
| **B2B SaaS** | Ayrshare, Buffer, Later 等 (有料自動投稿) | 月額 $50-500 | 一般ユーザーに高い |
| **AI モデル提供** | Midjourney, Runway, Pika (生成) | 月額 $20-100 | 品質アップグレード圧力 |
| **フリー系 Creator** | FLUX.2 [klein] + 自作スクリプト (完全無料) | 当PJ | 運用ノウハウ要求度高い |
| **Creator エージェンシー** | AI 動画制作代行 | Upwork 等 | 単価競争激化 |

### 視聴者トレンド
- **短尺化**: 動画全体の平均長 30秒→15-20秒へ急速短縮 (2024-2026)
- **AI への寛容度上昇**: 「AI生成 = 低品質」から「AI生成 = 新しい表現手法」へ評価変化
- **テーマ多様化**: 単なる「AI画像ギャラリー」から「ストーリー性のあるAI動画」需要へシフト

## プラットフォーム別 アルゴリズム攻略法

### YouTube Shorts

| 項目 | アルゴリズム特性 | 攻略法 |
|------|----------------|--------|
| **推奨度計算** | 発見度 (推奨度スコア順) > ホーム表示 | 新規ユーザー へのリーチを最大化. タグ・タイトルは検索性重視 |
| **完全視聴率** | 45-60秒は平均 40%, 30秒なら 65%, 15秒なら 75% | **当PJ戦略**: 30秒を中心に, テストで45秒も混ぜる |
| **エンゲージメント** | いいね・コメント (スクリーン時間より少ない重要度) | CTAで「いいね」明示 + 字幕で「コメント促進」メッセージ |
| **アップロード頻度** | 連続投稿ほど アルゴリズム有利（ただし上限 月100本推奨） | **当PJ**: 3本/日 (月90本) で最適ライン |
| **サムネイル** | 自動生成 (Shorts ネイティブ) で OK. カスタムは不要 | プラットフォームに任せ, 編集品質を優先 |
| **説明欄** | キャプション＋ リンク1個推奨. ハッシュタグ 5個程度 | 「詳細は YouTube チャネル / 説明欄」で誘導 |

### Instagram Reels

| 項目 | アルゴリズム特性 | 攻略法 |
|------|----------------|--------|
| **重視指標** | 完全視聴率 + **保存** (シェアよりも重い) | 「保存したくなる」実用性で設計 (教育, インテリア等) |
| **表示方法** | フィード内 + Explore Tab (ハッシュタグ, トレンド) | ハッシュタグ 20-30個で Explore 進出狙い |
| **投稿制限** | 1日 3本まで (ビジネスアカウント) | **当PJ**: 1本/日 に制限し, 品質優先 |
| **キャプション** | 最大 2,200文字. CTA必須 (フォロー, 保存促進) | 「このデザイン保存して後で試してみて!」など実用CTA |
| **クロスPF** | Facebook 自動転機で「二次リーチ」獲得 | Meta Business Suite で一括管理 |
| **音声** | Instagram Licensed Music Library 推奨 | Pixabay も OK (クレジット記載推奨) |

### TikTok

| 項目 | アルゴリズム特性 | 攻略法 |
|------|----------------|--------|
| **推奨度計算** | For You Page (FYP) = 完全視聴率 + エンゲージメント (加重平均) | 15-30秒の完全視聴率を最大化 |
| **完全視聴率** | 15秒: ~90%, 30秒: ~65%, 60秒: ~30% 目安 | **当PJ**: 15-30秒を中心に, 高速カット優先 |
| **エンゲージメント** | シェア (最重要) > コメント > いいね の順 |「友達に見せたい」エンタメ性を優先 |
| **投稿頻度** | 1日 1本 (開発者版 API 最大), ただし実際は 12時間以上推奨 | **当PJ**: API 制限で 1本/日に留まる可能性 → Instagram + YouTube で補完 |
| **アカウント年齢** | 3ヶ月未満は アルゴリズム低評価 | **当PJ**: TikTok アカウント事前作成を強く推奨 |
| **音声** | TikTok Sounds Library (流行音, トレンド効果) | Pixabay も OK だが, Sounds Library トレンド音を優先 |

### 共通攻略戦略
1. **完全視聴率最大化**: 動画長は短いほど有利 (15-30秒が黄金比)
2. **投稿頻度信号**: 最小 3本/日で「アクティブ」判定→アルゴリズム有利
3. **テーマ一貫性**: ブランド化で リピーターを獲得
4. **季節トレンド対応**: AI プロンプトを週次で トレンド最適化
