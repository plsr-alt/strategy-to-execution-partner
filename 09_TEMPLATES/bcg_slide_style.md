# BCGスライドスタイルガイド

参照元: https://www.kikakusyo.com/post/外資系コンサルティング会社-ボストンコンサルティング-の提案・調査資料実例10選
確認日: 2026-02-23
評価: 「踏襲の仕方は良い」（文字サイズは要調整）

---

## デザイン原則

| 項目 | ルール |
|------|--------|
| 背景 | **白背景**。装飾は最小限 |
| アクセントカラー | **グリーン系**（濃緑 #1A5C2A / 中緑 #2E8B45 / 薄緑 #A8D5B5） |
| タイトル | **結論・メッセージを文章形式で**（「〜している」「〜が課題だ」） |
| フォント | Yu Gothic / 游ゴシック |
| 文字サイズ | タイトル18pt以上・本文**10pt以上**（9pt以下は避ける） |
| 出所表記 | 左下に「出所：〇〇」小文字で記載 |
| バッジ | 「グッドプラクティス」「調査サマリー」等を緑帯で小さく |

---

## カラーコード（python-pptx用）

```python
GREEN_DARK  = RGBColor(0x1A, 0x5C, 0x2A)   # 濃緑（タイトル・強調）
GREEN_MID   = RGBColor(0x2E, 0x8B, 0x45)   # 中緑（バー・ヘッダー帯）
GREEN_LIGHT = RGBColor(0xA8, 0xD5, 0xB5)   # 薄緑（サブバー）
GREEN_PALE  = RGBColor(0xE8, 0xF5, 0xEC)   # 極薄緑（背景帯）
RED_ACCENT  = RGBColor(0xCC, 0x33, 0x33)   # 赤（CAGR等の強調数値）
```

---

## スライド構造テンプレート

```
[バッジ（任意）]  ← 緑帯に白文字8pt
[タイトル: 結論を文章で 18pt 濃緑 太字]
[サブタイトル: 10pt グレー]
──────────────────────── ← 緑ライン
[本文エリア]
  ・棒グラフ: 数値ラベルをバー上に直書き
  ・表: ヘッダー行を緑帯
  ・箇条書き: 左にアクセントライン（緑細帯）
──────────────────────── ← 薄グレーライン
出所：〇〇                          [ページ番号]
```

---

## 注意事項

- **紺（Navy）ヘッダー帯は使わない** → BCGスタイルではない
- タイトルは「市場規模」「主要プレイヤー」等の名詞NG → 文章形式で
- 文字が小さくなりがち → **本文最小10pt、キャプション最小8pt**
