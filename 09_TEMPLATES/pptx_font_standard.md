## スクリプト適用ルール（fix_fontsize_woodone.py 準拠）

```python
TARGET_SIZE = Pt(14)  # コンテンツスライド本文

# スキップするスライド（0-indexed）
SKIP_SLIDES = {
    0,   # 表紙
    1,   # 本資料の目的（中表紙）
    2,   # アジェンダ（中表紙）
    3,   # 背景・目的の振り返り（中表紙）
    6,   # ご報告（中表紙）
    12,  # 今後のご計画について（中表紙）
    14,  # 質疑応答（中表紙）
    15,  # 裏表紙
}

# スキップするplaceholder type
SKIP_PH_TYPES = {1, 3, 13}  # TITLE, CENTER_TITLE, SLIDE_NUMBER

# 20pt以上のテキスト（中表紙大見出し等）は変更しない
if run.font.size and run.font.size >= Pt(20):
    continue
```
