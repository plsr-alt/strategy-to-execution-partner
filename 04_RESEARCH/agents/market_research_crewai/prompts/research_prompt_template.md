# リサーチプロンプトテンプレート

## 用途
`crew.py` の Task description を手動でカスタマイズしたい場合の参考テンプレート。

---

## Researcher タスク（変更例）

```
「{topic}」について以下を Web 検索で調査・収集せよ。
1. 市場の定義・スコープ（何を含み、何を含まないか）
2. 市場規模データ（直近年・予測年・単位・出典URL）
3. 主要プレイヤー（企業名・特徴・シェア・出典URL）
4. 主要トレンド（技術・規制・消費者動向・出典URL）
5. 【追加】競合他社の価格帯・ポジショニング情報

各情報には必ず出典 URL を付記すること。
```

## Analyst タスク（変更例）

```
Researcher の調査結果をもとに、以下の観点で「{topic}」を深掘り分析せよ。
- Porter's Five Forces での業界構造分析
- プレイヤーを Tier1/2/3 に分類（基準: 売上・シェア・影響力）
- 3年後の市場変化シナリオ（楽観・中立・悲観）
```

---

## JSONスキーマ拡張例（implications を詳細化）

```json
"implications": [
  {
    "message": "ビジネス示唆・アクション",
    "priority": "High",
    "timeframe": "3ヶ月以内",
    "owner": "営業 / 技術 / マーケ"
  }
]
```
